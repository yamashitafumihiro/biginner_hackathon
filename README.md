# biginner_hackathon
初めてのハッカソンで制作したものです。

アルバイト等で不要の範囲を超えないように調整するのを補助するツールです。

ユーザーの属性や時給、週当たりのシフト数、その月の収入等を入力すると、労働状況へのアドバイスが返ってきます。

This is a product from the first hackathon I participated in.

This tool assists in adjusting part-time jobs to ensure that they do not exceed the scope of support.

Enter the user's demographics, hourly wage, number of shifts per week, income for the month, etc., and the system will return advice on the work situation.
